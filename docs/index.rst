Welcome to the Yakkl documentation!
===================================

Welcome! Yakkl's documentation is split into four parts:

- `User documentation <https://yakkl.com/help>`__, for users and
  administrators of Yakkl organizations.
- `Installation documentation <production/install.html>`__, for
  installing and maintaining a production self-hosted Yakkl installation.
- `API documentation <https://yakkl.com/api/>`__, for writing
  integrations or bots using the Yakkl API.
- `Contributor documentation <overview/contributing.html>`__, for
  developing the Yakkl software, translating, submitting bug reports,
  or making other contributions to the project.

Yakkl has well over 150,000 words of documentation. If you can't find
what you're looking for, please `let us know
<contributing/chat-yakkl-org.html>`__!  Further information on the
Yakkl project and its features can be found at `https://www.yakkl.org
<https://www.yakkl.org>`__.

This site contains our installation and contributor documentation. If
this is your first time here, you may want to start with `Production
installation <production/install.html>`_ or `Contributing to Yakkl
<overview/contributing.html>`_.
