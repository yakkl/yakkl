# Installation instructions

Yakkl's REST API is easy to work with directly, but there are API
libraries available for a few popular languages.

The Python library is the most advanced (and has tools for easily
writing interactive bots that react to messages), so we recommend it
if you're trying to decide.

{start_tabs}
{tab|python}

Install the Python API with [pip](https://pypi.python.org/pypi/yakkl/):

```
pip install yakkl
```

{tab|yakkl-send}

Included with the Python bindings:

```
pip install yakkl
```

{tab|js}

Install the JavaScript API with [npm](https://www.npmjs.com/package/yakkl-js):

```
npm install yakkl-js
```

{tab|curl}

No download required!

{end_tabs}
