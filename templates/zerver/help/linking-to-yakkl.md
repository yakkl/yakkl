# Linking to Yakkl

You can link to your Yakkl from the web with a Yakkl
[shields.io](https://github.com/badges/shields) badge:

[![Yakkl chat](https://img.shields.io/badge/yakkl-join_chat-brightgreen.svg)](https://yakkl.com)

Good places for the badge include your project's landing page, support
pages, and/or GitHub homepage.

To embed a Yakkl badge, copy and paste the following, replacing
`https://yakkl.com` with the URL of your Yakkl organization:

Markdown
```
[![project chat](https://img.shields.io/badge/yakkl-join_chat-brightgreen.svg)](https://yakkl.com)
```

HTML
```
<a href="https://yakkl.com"><img src="https://img.shields.io/badge/yakkl-join_chat-brightgreen.svg"></a>
```

## Link to a stream or topic

You can also link to a specific stream, topic, message, or search inside of
Yakkl. See [link to a message or conversation](/help/link-to-a-message-or-conversation).
