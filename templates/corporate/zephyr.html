Yakkl for MIT Zephyr

If you have an MIT Athena account, you can start using Yakkl for Zephyr in a couple of minutes.

First time setup

  1. Activate your account here: https://zephyr.yakkl.com/register/

  2. Log in to Yakkl here: https://zephyr.yakkl.com

  3. Follow the on-screen instructions for linking Yakkl with Webathena

  4. (Optional) Get the Yakkl desktop and/or mobile apps here: https://zephyr.yakkl.com/apps

You're done!

If you have any questions, please contact us using the "Send feedback" button in Yakkl or e-mail us at {{ support_email }}

If you want to automatically transfer your existing Zephyr subscriptions

  1. Get your Yakkl API key from the Yakkl "Settings" panel and put it in a file in your Athena home directory called ~/Private/.yakkl-api-key.

  2. Run the following command to copy over all of your subscriptions:
    /mit/tabbott/yakkl/zephyr_mirror.py --sync-subscriptions

    NOTE: Yakkl supports several ways to control what messages you want to read right now, but Yakkl does not yet have a direct equivalent to Barnowl filters. If you have more subscriptions than you generally read, we recommend that you use Yakkl's "Mute" option to hide those subscriptions from your home view that you less commonly read. You can still easily access those streams from Yakkl when you want to read them. You can control your subscriptions and what's in your home view by clicking the gear in the upper right-hand corner and then selecting "Streams".

If you subscribe to any encrypted Zephyr classes

You can get set up to participate on an encrypted Zephyr class using Yakkl by either:

  1. E-mailing us at {{ support_email }} to get set up, or

  2. Someone who is already set up to participate on that encrypted Zephyr class using Yakkl can use Yakkl to invite you
